Introduction to Arrays

  • Arrays are used to store multiple values in a single variable.
  • Arrays are a type of object in JavaScript and can hold values of any data type.
  • Arrays use zero-based indexing, meaning the first element is at index 0.

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // Output: Apple

Creating Arrays

  • Arrays can be created using the array literal syntax or the Array constructor.

// Using array literal
let colors = ["Red", "Green", "Blue"];

// Using Array constructor
let numbers = new Array(1, 2, 3, 4);

Accessing Array Elements

  • Access elements using their index (position) in the array.
  • Array indices start at 0.

let cars = ["Toyota", "Ford", "Tesla"];
console.log(cars[1]); // Output: Ford

// Changing an element
cars[1] = "BMW";
console.log(cars); // Output: ["Toyota", "BMW", "Tesla"]

Array Properties and Methods

  • length: Returns the number of elements in the array.
  • push(): Adds one or more elements to the end of the array.
  • pop(): Removes the last element from the array.
  • shift(): Removes the first element from the array.
  • unshift(): Adds one or more elements to the beginning of the array.

let numbers = [1, 2, 3, 4];
console.log(numbers.length); // Output: 4

numbers.push(5); // Adds 5 to the end
console.log(numbers); // Output: [1, 2, 3, 4, 5]

numbers.pop(); // Removes the last element
console.log(numbers); // Output: [1, 2, 3, 4]

Iterating Over Arrays

  • Use loops to iterate over arrays: for, for...of, and forEach().

let fruits = ["Apple", "Banana", "Cherry"];

// Using for loop
for (let i = 0; i < fruits.length; i++) {
    console.log(fruits[i]);
}

// Using for...of loop
for (let fruit of fruits) {
    console.log(fruit);
}

// Using forEach method
fruits.forEach((fruit) => console.log(fruit));

Common Array Methods

  • map(): Creates a new array with the results of calling a function on every element.
  • filter(): Creates a new array with elements that pass a test function.
  • reduce(): Reduces the array to a single value by executing a function on each element.
  • find(): Returns the first element that passes a test function.
  • includes(): Checks if an array contains a specified element.

let numbers = [1, 2, 3, 4, 5];

// map
let doubled = numbers.map(num => num * 2);
console.log(doubled); // Output: [2, 4, 6, 8, 10]

// filter
let evens = numbers.filter(num => num % 2 === 0);
console.log(evens); // Output: [2, 4]

// reduce
let sum = numbers.reduce((total, num) => total + num, 0);
console.log(sum); // Output: 15

Multi-Dimensional Arrays

  • Arrays can contain other arrays, creating multi-dimensional arrays.
  • Access elements using multiple indices.

let matrix = [
    [1, 2, 3],
    [4, 5, 6],
    [7, 8, 9]
];

console.log(matrix[1][2]); // Output: 6

Destructuring Arrays

  • Destructuring allows you to unpack values from arrays into distinct variables.
  • Introduced in ES6, it simplifies array element assignment.

let colors = ["Red", "Green", "Blue"];
let [first, second, third] = colors;

console.log(first); // Output: Red
console.log(second); // Output: Green

Spread and Rest Operators

  • Spread: Expands elements of an array into individual elements.
  • Rest: Collects all remaining elements into an array.

// Spread operator
let numbers = [1, 2, 3];
let moreNumbers = [...numbers, 4, 5];
console.log(moreNumbers); // Output: [1, 2, 3, 4, 5]

// Rest operator
function sum(...args) {
    return args.reduce((acc, num) => acc + num, 0);
}

console.log(sum(1, 2, 3, 4)); // Output: 10

Common Mistakes with Arrays

  • Confusing array indices and values.
  • Mutating arrays unintentionally (e.g., using push() when not desired).
  • Using the wrong method for iteration (e.g., modifying the array during forEach()).

let numbers = [1, 2, 3];
numbers[3] = 4; // Correct way to add
numbers[5] = 6; // Leaves undefined gaps

console.log(numbers); // Output: [1, 2, 3, 4]

Best Practices

  • Use array methods like map(), filter(), and reduce() for cleaner, more functional code.
  • Avoid using var with arrays; prefer let or const for scope and immutability.
  • Use destructuring for clearer and more concise code.

Summary

  • Arrays are essential for handling collections of data in JavaScript.
  • Understanding array methods and properties enhances your ability to manipulate data effectively.
  • Best practices help in writing clean, efficient, and maintainable code using arrays.
